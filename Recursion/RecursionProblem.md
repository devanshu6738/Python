🟢 Beginner (build basic intuition)
1. Factorial

Problem:
Compute n! = n × (n-1) × ... × 1

👉 Key idea: break into smaller subproblem

fact(n) = n * fact(n-1)
2. Fibonacci

Problem:
Find the nth Fibonacci number.

fib(n) = fib(n-1) + fib(n-2)

⚠️ Good for understanding recursion tree (but inefficient)

3. Sum of array

Problem:
Find sum of elements in an array using recursion.

4. Reverse a string

Problem:
Reverse a string recursively.

5. Power (xⁿ)

Problem:
Compute x^n recursively.

🟡 Intermediate (patterns start forming)
6. Check palindrome

Problem:
Check if a string is a palindrome using recursion.

7. Binary Search (recursive)

Problem:
Implement binary search using recursion.

8. Print all subsets (Power set)

Problem:
Given array [1,2,3], print all subsets.

👉 Pattern: include / exclude

9. Tower of Hanoi

Problem:
Move disks between rods following rules.

👉 Classic recursion thinking problem

10. GCD (Euclidean algorithm)
gcd(a, b) = gcd(b, a % b)
🔵 Advanced (important for interviews)
11. Permutations of a string

Problem:
Print all permutations of a string.

👉 Pattern: swapping / backtracking

12. Combination sum

Problem:
Find all combinations that sum to a target.

👉 Used heavily in coding interviews

13. N-Queens

Problem:
Place N queens on board safely.

👉 Classic backtracking

14. Sudoku Solver

👉 Recursive + constraint checking

15. Word Search (Grid DFS)

👉 Explore grid using recursion
